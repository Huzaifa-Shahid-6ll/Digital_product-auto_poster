## User Flow (Context from CONTEXT.md)

1. **Input niche** → User enters niche keyword
2. **View ideas** → See 3+ AI-generated product ideas with rationale
3. **Select idea** → User clicks to select one idea
4. **Generate product** → System creates PDF from selected idea
5. **Quality check** → AI validates completeness, formatting, coherence
6. **Human review** → User reviews product, approves or requests revision
