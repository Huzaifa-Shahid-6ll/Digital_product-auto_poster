Endpoints:
- POST /api/reviews/ideas - Submit ideas for review, returns review_id
  - Body: {niche: str, ideas: list[ProductIdea]}
  - Returns: {review_id, state: "IDEAS_GENERATED", ideas: [...]}

- POST /api/reviews/select - User selects an idea, triggers product generation
  - Body: {review_id: str, selected_idea_id: str}
  - Returns: {review_id, state: "PRODUCT_GENERATED", product: ProductOutput, validation: ValidationResult}

- POST /api/reviews/approve - User approves the product
  - Body: {review_id: str}
  - Returns: {review_id, state: "APPROVED", product_id, next_steps: "Ready for Phase 3"}

- POST /api/reviews/reject - User rejects, returns to idea generation
  - Body: {review_id: str, feedback: str}
  - Returns: {review_id, state: "IDEAS_GENERATED"}

- GET /api/reviews/{review_id} - Get current review state and contents
